# Reportify 

Данный продукт предназначен для удобного создания аналитических табличных и графических отчетов, исключяя из работы создания sql-запросов для получения данных из базы

## Зависимости

Для развертывания системы на своем устройстве вам необходимо заполнить/изменить обязательные поля в файле .env. Обязательные поля помечены стрелками "<---"

## Быстрый запуск

Для запуска необходим docker. Снизу находится инструкция для запуска приложения на linux + docker

Само приложение находится на: localhost:1337

```
cd reportify/docker

-- Если у вас есть makefile --
make app-up

-- Если у вас нет makefile --
docker compose up -d 

```

## Особенности защиты

* Пароли все хэшируются через BCrypt
* Секретные ключи обновляется каждый раз, когда вы полностью перезапускаете приложение
* Время жизни токенов вы можете настроить в файле конфигурации
