# Reportify

Reportify — университетский проект для автоматизации генерации отчетов из SQL-запросов. Клиент отправляет запрос и формат, сервис создает задачу, воркер собирает данные из источника, конвертирует результат и публикует готовый файл по ссылке.

## Статус проекта

Проект находится в активной разработке. Часть фронтенда еще в процессе реализации, а тесты находятся на стадии внедрения и будут расширяться по мере стабилизации функционала.

## Основные возможности

- создание отчетов и отслеживание статуса генерации
- форматы: PDF, XLSX, CSV, JSON, DOCX
- хранение и выдача файлов через MinIO с presigned-ссылками
- кэширование статусов в Redis
- управление пользователями и ролями, JWT-аутентификация
- REST gateway с Swagger и внутренние gRPC-сервисы
- асинхронная обработка задач через Kafka и DLQ
- просмотр схем, таблиц и колонок источника данных

## Архитектура сервисов

- gateway — HTTP API, авторизация, Swagger
- auth — управление пользователями/ролями и токенами
- report — создание задач и метаданные отчетов
- report-worker — обработка задач, конвертация и публикация файлов

## Технологии

Go, Gin, gRPC, PostgreSQL, Redis, Kafka, MinIO, Docker, Swagger.

## Быстрый старт (Docker)

1. Скопируйте `.env.example` в `.env` и заполните значения.
2. Запустите инфраструктуру и сервисы:

```bash
cd deployment
make up
```


Nginx: `http://localhost:8088` (статические файлы/фронтенд, в разработке)
